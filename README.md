# 📐 Cálculo Numérico (CNUM)

Implementações de métodos numéricos desenvolvidas em Java durante a disciplina de Cálculo Numérico na UFERSA.

## 🛠️ Tecnologia

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=openjdk&logoColor=white)

## 📂 Métodos Implementados

**Zeros de Funções**
- Método da Bissecção
- Método da Falsa Posição
- Método de Newton-Raphson
- Método da Secante

**Sistemas Lineares**
- Eliminação de Gauss
- Fatoração LU
- Métodos iterativos: Jacobi e Gauss-Seidel

**Integração Numérica**
- Regra dos Trapézios
- Regra de Simpson 1/3
- Soma de Riemann

## ▶️ Como executar

### Pré-requisitos

- Java 8+
- Eclipse IDE (recomendado)

### 1. Clonar o repositório

```bash
git clone https://github.com/ErkPatrick/CNUM.git
cd CNUM
```

### 2. Importar no Eclipse

**File → Import → Existing Projects into Workspace** e selecione a pasta do projeto.

Execute a classe desejada clicando com o botão direito → **Run As → Java Application**.
